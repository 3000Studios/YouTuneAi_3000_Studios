== Description ==

YouTuneAi is a cutting-edge WordPress theme designed specifically for AI-powered music and content creation platforms. With built-in WooCommerce support, custom post types for AI features, and comprehensive revenue generation capabilities, this theme is perfect for creators, musicians, and digital entrepreneurs.

== Installation ==

1. Upload the theme files to the `/wp-content/themes/youtuneai-theme` directory
2. Activate the theme through the 'Appearance' > 'Themes' menu in WordPress
3. Configure theme options via Appearance > Customize
4. Install recommended plugins (WooCommerce for shop functionality)
5. Import demo content (optional)

== Frequently Asked Questions ==

= Does this theme require WooCommerce? =

No, WooCommerce is optional. However, to use shop functionality, you need to install and activate WooCommerce.

= Is the theme translation ready? =

Yes, the theme is fully translation ready with a .pot file included.

= Can I use this theme for commercial purposes? =

Yes, the theme is licensed under MIT, which allows commercial use.

= Does it work with page builders? =

Yes, the theme is compatible with popular page builders like Elementor, Beaver Builder, and Gutenberg.
